# AgenTester 

Automated User-Agent Tester with *proxy support.

> Do you trust User-Agent? Don't… 😏

---

## 🔥 Features
- Test multiple User-Agents from a wordlist
- Multi-threaded for faster results ⚡
- Proxy support (Burp Suite / OWASP ZAP)
- Colored output for better visibility
- Simple and lightweight

---

## 📦 Requirements

- Python 3
- requests
- colorama
- urllib3

Install dependencies:

```bash
pip install -r requirements.txt
